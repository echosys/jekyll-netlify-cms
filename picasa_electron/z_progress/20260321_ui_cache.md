# March 21, 2026 - UI Cleanup & Cache Clear Feature

## Completed Tasks

### 1. Fixed .app Folder Handling ✓
- Modified `scanner_worker.ts` to properly skip `.app`, `.framework`, and `.bundle` directories
- Changed logic to exclude packages entirely instead of treating them as files
- Now only indexes image and video files (`.jpg`, `.png`, `.mp4`, etc.)
- `.app` bundles are now added to the discovered files list as type 'file' (opaque, single unit)

### 2. Added Clear Cache Feature ✓
- Added `clear-cache` IPC handler in `main.ts` that clears:
  - `scan_results` table
  - `thumbnails` table
  - `scan_cache` table
- Added "Clear Cache" menu option in File menu
- Exposed `clearCache()` API in preload
- Implemented cache clear handler in renderer that:
  - Clears photos and thumbnails from state
  - Auto-rescans all active workspaces

- **Buttons**: Added border-radius, transitions, active states with shadows, padding, font sizing

- **Cards**: Enhanced shadows and hover effects
- **Sidebar**: 
  - Improved spacing (more padding between sections)
  - Better visual hierarchy
  - Refined hover states
  - Larger checkbox and icons
- **Spacing**: Added consistent margin utilities (mb-1, mb-3, mt-1, gap-6)
- **General**: Better transitions, refined colors, improved UX feedback

### 4. Photo Grid Layout & Text Sizing ✓
- **Photo Grid**: CSS Grid layout with responsive columns
  - 140px minimum tile width, auto-fill creates responsive columns
  - 12px gaps between tiles with proper padding
  - Responsive breakpoints for different screen sizes
